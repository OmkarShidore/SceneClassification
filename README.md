# SceneClassification
## Goal
The goal is to classify around 25k images of size 150x150 distributed under 6 categories.
Categories- 0:Building, 1:Forest, 2:Forest, 3:Glacier, 4:Mountain, 5:Street

## Framework
Pytorch v-1.3.1

## Transfer Learning
Model: AlexNet
